# Pygame Typing Game

A clone of a clone of a typing game I came across itch. Originally used it as an exercise to learn the [LÃ–VE](https://love2d.org/) framework, now using it to learn pygame.

## How To Play

-   Type a word the starts with the letter on screen and press enter to submit the word. The last letter of the submitted work will be the first letter of the next word.
